# Issue 076: Implement 30-hour core completion and 100% tracking

**Wave**: wave-5
**Milestone**: M5 Store Expansion
**Labels**: `gameplay`, `progression`, `phase:m4plus`, `priority:high`
**Dependencies**: issue-036, issue-054

## Why This Matters

Completion tracking gives the game its end-state and validates the 30-hour target.

## Scope

Track core completion criteria. Track 100% completion criteria. Display progress in menu. Trigger completion ending when all criteria met. Completion data in save file.

## Deliverables

- Core completion checklist (from PROGRESSION.md)
- 100% completion checklist (all items seen, all stores maxed, all milestones)
- Progress percentage visible in save slot and menu
- Ending trigger at 100%
- Completion stats in save data

## Acceptance Criteria

- Core completion achievable in ~30 hours
- 100% is significantly more
- Progress tracking is accurate
- 100% triggers ending sequence
