# Issue 066: Implement accessibility features

**Wave**: wave-4
**Milestone**: M4 Polish + Replayability
**Labels**: `ux`, `ui`, `phase:m4plus`, `priority:medium`
**Dependencies**: issue-027

## Why This Matters

Accessibility is not optional. It's required for a shippable product.

## Scope

Key rebinding, UI scaling options, colorblind-friendly rarity indicators (shapes not just colors), configurable text size.

## Deliverables

- Key rebinding UI in settings
- UI scale slider (0.8x - 1.5x)
- Rarity indicators use shapes + colors
- Text size options
- All settings persist

## Acceptance Criteria

- Can rebind all gameplay keys
- UI scales correctly at all sizes
- Rarity distinguishable without color
- Text remains readable at all sizes
