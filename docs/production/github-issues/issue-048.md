# Issue 048: Implement upgrade system for store fixtures

**Wave**: wave-3
**Milestone**: M3 Progression + Content Expansion
**Labels**: `gameplay`, `progression`, `phase:m3`, `priority:medium`
**Dependencies**: issue-047, issue-010

## Why This Matters

Upgrades give the player meaningful ways to invest profit.

## Scope

Buy better shelving, display cases, signage from a fixture catalog. Each upgrade costs money. Better fixtures display more items or attract customers. Tied to store progression.

## Deliverables

- Fixture catalog with prices
- Purchase fixture: deduct cash, add to store
- Upgraded fixtures: more slots, better display (affects customer purchase probability)
- Signage: affects foot traffic multiplier

## Acceptance Criteria

- Can buy fixtures from catalog
- Better shelf has more item slots
- Signage increases customer spawn rate
- Fixtures persist in save file
