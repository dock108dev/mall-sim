**Wave**: wave-5

**Labels**: `testing`, `production`, `phase:m4plus`, `priority:high`

## Why This Matters

The owner wants to beat the game 3-5 times before release. This issue is that commitment.

## Scope

3-5 full playthroughs by the owner. Each playthrough validates: core completion achievable, economy balance, progression flow, save/load reliability, no crashes. Document issues found.

## Deliverables

- 3-5 complete playthrough logs
- Bug list with reproduction steps
- Economy balance notes
- Progression pacing notes
- Save/load reliability confirmation
- Each playthrough: different starting store, different strategy

## Acceptance Criteria

- 3 full playthroughs completed without game-breaking bugs
- Core completion achieved in each playthrough
- Economy feels balanced across tested stores
- Save/load never corrupts data
- Documented issues have fixes or accepted-risk status
